What is SOTAwatch ?

Summits on the Air (SOTA) is an award scheme for radio amateurs and shortwave listeners that encourages portable operation in mountainous areas. "Activators" go to the summits with their radio, and "chasers" try to contact them. Your will find more information on http://www.sota.org.uk/

SOTAwatch is a informational web site that helps chasers to know when and where the activators are. You will find more information on http://www.sotawatch.org/ .

What is SOTAwatch filter ?

SOTAwatch filter is a web application that takes the "spots" as published on www.sotawatch.org, and makes it possible to filter them depending on calls-signs, summits, bands, modes, comments and much more. And all you need is a recent internet browser running on any recent device (PC, smartphone, tablet, etc.)

How to filter spots ?

To filter spots, head over to the settings tab. Under the filter settings you have all the options to filter out spots based on multiple criteria.

To filter out specific callsigns or summits we use regular expressions. If you fill in the "include" textbox only spots that match the regular expression inside the textbox will be showed. If you fill in the "exclude" textbox all the matching spots will be hidden. Of course you can use the two together for maximum flexibility.

Examples
Regexp Filters
(ON4ABC|ON4DEF|ON4GHI) Only ON4ABC or ON4DEF or ON4GHI
^(DL|DM).*Beginning with DL or DM
^[^NWA].*Not beginning with N or W or A
^F/(AM|AB).*Summits: French Alps only
^EA8.* Summits: Canary Islands only

On SOTAwatch filter, the case sensitivity of the Regular Expressions has been turned OFF: "SsB" is therefore the same as "ssB" or "SSB"

And finally, I want to share a useful ressource: Regex101 with this tool you can check if your regular expression matches what you wanted it to match. Very useful!

NoScript or other JavaScript blocking plug-ins intalled?

If you have NoScript installed or any other JavaScript blocking plug-in you should whitelist at least on6zq.be and mathieudavid.org. Because SOTAwatch filter is hosted on the latter and uses a script on on6zq.be to fetch the spots. If you don't allow at least those 2 domains to run scripts then I am afraid that SOTAwatch filter will not work correctly.

It's not working, what should I do?

SOTAwatch filter relies heavily on JavaScript and uses the AngularJS framework. Older browsers do not always fully support those technologies. We have experienced trouble with older Internet Explorers and older Android browsers.

If SOTAwatch filter does not seem to work properly, please update your browser.

We recommend Google Chrome because it is the browser we use in the development and testing phase, but Safari, Firefox and Opera should work equally well if they are up to date. We advice against Internet Explorer especially the older versions (before Internet Explorer 9).
